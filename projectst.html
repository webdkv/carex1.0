Currator Statement

Tantan Nugraha, S.Pd.

This exhibition is the final project of visual communication design students at vocational schools. This exhibition is themed "basic art and digital design" because the designs we make are based on the main material of visual design at vocational schools. We have completed 4 main materials such as interactive media design, web design and development, animation and creative projects and entrepreneurship.

In this interactive media design course, we learn about the basics of human interaction with computers, UI/UX processes, how to create UI/UX prototypes and usability testing for UI/UX prototypes. In this web design and development course, we learn about the basics of web design with HTML and CSS, the basics of web development with Javascript, creating web designs with WordPress and creating web designs with Bootstrap. In this animation course, we learn about basic art and design for animation such as basic drawing, gesture drawing, perspective drawing, nirmana and character rotation sheets. In addition, we also learn to make stop motion, 2D animation and 3D modeling. In this creative project and entrepreneurship course, we learn about visual branding, understanding products and creating mockup packaging designs.

In creating this artwork, students use traditional and digital techniques. We use various tools and software such as figma, visual studio code, capcut, etc. From the various techniques and tools used, we produce many artworks such as facial illustrations, mockups, UI/UX prototypes, characters and sketches, basic drawings, nirmana, stop motion, web design, 2D animation, and 3D models.

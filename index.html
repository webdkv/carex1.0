Caringin Art Exhibition 1.0

CAREX 1.0 is a mini exhibition that aims to appreciate the artwork of vocational student in major visual communication design batch 2023

In this exhibition we have many diverse artwork such as face illustration, UI/UX wireframe, UI/UX prototype, character and sketch, basic drawing, nirmana, stop motion, web design plugin, 2D short animation, 3D model and web design bootstrap

About Subject Material

Interactive Media Design

In this subject we learn about foundation of human computer interaction, UI/UX process, how to create UI/UX prototype and usability testing for UI/UX prototype

Web Design and Development

In this subject we learn about foundation web design with HTML and CSS, foundation web development with Javascript, create web design with WordPress and create web design with Bootstrap

Animation

In this subject we learn about basic art and design for animation such as basic drawing, gesture drawing, perspective drawing, nirmana and character rotation sheet. Beside that, we also learn create stop motion, 2D animation and 3D modeling

Creative Project and Entrepreneurship

In this subject we learn about visual branding, understanding about product and create mockup packaging design
